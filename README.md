#  Sales Forecasting Dashboard

This project presents a Power BI dashboard that compares actual sales with forecasted values.

##  Features

* Sales vs Forecast trend analysis
* KPI cards for total sales and forecasted sales
* Forecast error evaluation
* Interactive date filtering

##  Tools Used

* Power BI
* Python (for forecasting)
* Pandas, Matplotlib

## Insights

* Sales show an upward trend with seasonal variations
* Forecast provides a baseline for future planning
* Error metrics help evaluate prediction accuracy

##  Files

* `.pbix` → Dashboard
* `.csv` → Dataset
